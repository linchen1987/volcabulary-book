# Vocabulary Book

单词本应用 - 记录和学习英语单词

## 功能

### 1. 单词管理
- 添加、编辑、删除单词
- 支持单词、短语、句子
- 记录翻译、解释、例句
- 音标支持

### 2. 学习辅助
- Level 级别标记掌握程度
- 单词筛选和排序
- 测验功能（计划中）
- AI 辅助生成（计划中）

### 3. 数据独立
- 数据存储在浏览器 IndexedDB
- 支持 WebDAV 和 S3 同步
- 数据导入导出

### 4. PWA 支持
- 每个空间可独立安装为 APP
- 离线使用

## 技术栈

- React 19
- React Router 7
- Dexie (IndexedDB)
- Tailwind CSS v4
- Cloudflare Workers

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建
pnpm build

# 类型检查
pnpm typecheck

# 代码检查
pnpm lint
```
