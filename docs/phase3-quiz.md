# Phase 3: 测验功能

## 目标

实现单词测验功能，帮助用户记忆单词。

---

## 1. 测验流程

```
开始测验 → 随机选择单词 → 显示单词 → 用户思考 → 翻开答案 → 用户自评 → 更新 level
```

---

## 2. 核心功能

### 2.1 测验设置
- [ ] 选择测验范围（全部 / 特定 level）
- [ ] 设置测验数量
- [ ] 开始测验按钮

### 2.2 测验进行
- [ ] 显示单词（content）
- [ ] "显示答案"按钮
- [ ] 显示完整信息
- [ ] "通过" / "不通过" 按钮
- [ ] 进度指示器

### 2.3 结果处理
- [ ] 不通过：level + 1
- [ ] 首次通过：level - 1 (最低为 0)
- [ ] 测验完成统计

---
